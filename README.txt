Οδηγίες Τρεξίματος


1. 3D R-tree 

Για να τρέξει κανείς λίγα δεδομένα από το data set για γρήγορα αποτελέσματα τρέχει το αρχείο rtree_index.py.

Για να τρέξει κανείς όλο το data set για τα αποτελέσματα και τα διαγράμματα τρέχει το αρχείο rtree_index.py.


2. Interval και Segment Tree

Για το Interval Tree, αρχικά πρέξει να τρέξει το interval_tree.py αρχείο και έπειτα το αρχείο main.py.

Για το Segment Tree, αρχικά πρέξει να τρέξει το segment_tree.py αρχείο και έπειτα το αρχείο main1.py.


3. Convex Hull, Skyline Operator

Εγκατάσταση data set από : https://www.kaggle.com/datasets/mustafasayed1181/movies-metadata-cleaned-dataset-19002025

Για την εκτέλεση των αρχείων κώδικα απαιτείται το αρχείο "data_movies_clean.xlsx" να βρίσκεται στον ίδιο φάκελο με τα αρχεία πηγαίου κώδικα.

απαιτείται η εγκατάσταση των παρακάτω βιβλιοθηκών:
-pandas & openpyxl
-numpy
-matplotlib
-scipy

εγκατασταση με την εντολή: pip install pandas numpy matplotlib scipy openpyxl

η εκτέλεση μπορεί να διαρκέσει λίγα λεπτά, θα εμφανιστούν μετρήσεις στην κονσόλα και γραφήματα.


4. Line Segment Intersection 

Για να ζητήσει ο χρήστης τον αριθμό από τα segments που θα δημιουργηθούν (προτείνεται να ζητήσει από 5000 εώς 90000), και για να δει τους χάρτες των τομών, τρέχει το αρχείο Line_Segment_Intersection.py

Για να δει ο χρήστης τη χρονική διαφορά του Brute Force και Bentley-Ottmann τρέχει το LIS_analysis.py.
